FireCull

Cull faster — without changing your Lightroom workflow. FireCull speeds up your first pass by ranking sharpness and flagging closed eyes. You stay in control.

FireCull is designed to speed up your first pass so you spend less time zooming in and more time delivering.

What it does

  • Built for wedding & event photographers culling thousands of images per shoot.
  • Applies star ratings in Absolute or Relative mode to surface stronger frames faster.
  • Flags likely closed-eye shots so you can catch obvious misses quickly.
Quick demo
Run FireCull from Lightroom and open the report in seconds.

How to use

  • Install plugin → open Lightroom Classic
  • Go to File → Plug-in Extras → Analyze with FireCull
  • Review the ratings and keep moving

Apply selections automatically

FireCull can export your report selections as a .json file, then apply them back in Lightroom as labels, pick flags, or a collection.

  • Select photos in the FireCull report
  • Click Export Selected (.json)
  • Go to File → Plug-in Extras → Apply Selection (FireCull Report)
Export selections from the report, then apply them back in Lightroom in a few clicks.
